1 como iniciar o git ?
R: git init 
2 como identificar os arquivos modificados ?
R: git status
3 como adicionar os arquivos no meu pacote ?
R: git add .
4 como organizar os arquivos pra mandar pro servidor ?
R: git commit -m "nome do arquivo" 
 5 como eu envio os pacotes pro meu servidor ?
 R: git push -u origin main
